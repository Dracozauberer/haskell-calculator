## About

This is a learning project to demonstrate functional programming concepts in Haskell, including:
- Algebraic Data Types (ADTs)
- Pattern matching
- Recursion
- Parsing
- Pure functions

The calculator can evaluate mathematical expressions like `2 + 3 * (4 - 1)` with proper operator precedence.
